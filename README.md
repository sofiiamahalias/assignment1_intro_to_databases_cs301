# assignment1_intro_to_databases_cs301
Я створила 5 таблиць з описом шести студентів: main - складається з унікального id, імені, прізвища та пошти; study_programs - field (галузь знань), середня оцінка на курсі (avarage_grade), найлегший та найскладніший предмети; hobby_details - містять кількість років, скільки займаються (years_number), приблизну кількість отриманих за цей час медалей (approximate_medals_number) та орієнтовний час витрачений на хобі в годинах (hours_spent_for_hobby); study_general_info - загальна інформація про навчання (id студента, рік вступу, місто вступу, назва університету та направлення); hobbies - має id, назва активності, рівень людини (професіонал чи аматор) та роки занять.
